# **VTP and trunk links** #

These scripts are the configuration of the switches ( Access and Distribution layer)

#Commands#

  *1* Show vtp status
  
  *2* Show interface <int> switchport
  
  *3* Show interface trunk
  
  *4* Show vlan brief | incl active
  
  *5* Show vtp password (version 1 and 2)
  
  *6* Show vlan brief
